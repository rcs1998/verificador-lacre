📱 Como abrir o sistema
Abra o Chrome (Android) ou Safari (iPhone) e acesse: rcs1998.github.io/verificador-lacre

Para salvar como atalho na tela inicial: no Chrome, toque nos 3 pontinhos → "Adicionar à tela inicial". No Safari, toque em compartilhar → "Adicionar à tela de início".
🔑 Login normal
1. Digite seu e-mail cadastrado pelo administrador
2. Digite sua senha
3. Pressione Enter ou toque em ENTRAR
🔑 Esqueci a senha
1. Digite seu e-mail no campo
2. Toque em "🔑 Esqueci minha senha"
3. Verifique sua caixa de entrada — chegará um link por e-mail
✅ Após o primeiro login online, o sistema funciona offline por 7 dias. Uma faixa laranja aparece no topo quando estiver sem internet.
⚠️ "Usuário inativo" = conta bloqueada. Contate o administrador.
Aba Origem — Tela Real do Sistema
Tela exibida após o login — preencha os dados antes de fotografar o lacre
O que é cada campo
Operador (canto superior esquerdo) — exibe seu nome. Confirme que é você antes de registrar
Botões Admin / Sair / Ajuda (canto superior direito) — Admin abre o painel de gestão; Sair encerra sua sessão; Ajuda abre este manual
Abas ORIGEM / DESTINO — ORIGEM (azul) para registrar saída; DESTINO para verificar chegada. Toque para trocar
Placa do Veículo — digite sem hífen (ABC1234 ou ABC1D23). Ao sair do campo o sistema valida automaticamente: ✅ verde = autorizada, ❌ vermelho = não cadastrada
Motorista — nome completo do motorista. Campo obrigatório
Observações — campo opcional para anotações. Deixe em branco se não houver nada a registrar
⚠️ Role a tela para baixo após preencher esses campos — o card da câmera do lacre aparece logo abaixo.
✅ Placa com ✅ verde = tudo certo, pode continuar. ❌ Vermelho = chame o administrador para cadastrar a placa.

✅ IA leu com sucesso
A câmera detecta o QR Code e tira a foto automaticamente. A IA lê o número e verifica se o lacre está íntegro. O resultado aparece logo abaixo em verde (íntegro) ou vermelho (violado).
⌨️ IA não conseguiu ler
Um campo de entrada manual aparece automaticamente. A foto já foi registrada — basta digitar o número que está impresso fisicamente no lacre. O processo nunca trava por causa da IA.
✅ A foto é sempre registrada como evidência antes de qualquer análise da IA.
💡 Dicas para uma boa foto
• Aponte a câmera diretamente para o QR Code — sem ângulo lateral
• Mantenha entre 10 e 30 cm de distância
• Verifique se há boa iluminação — evite contra-luz
• Segure o celular firme — movimento borrará a imagem
• O indicador na tela mostra quando o QR Code está sendo detectado
Nota Fiscal
Como fotografar a NF e confirmar os dados do fornecedor
📋 O card da NF aparece automaticamente
Após o lacre ser verificado, um aviso azul aparece indicando "PASSO 2 — NOTA FISCAL". Role a tela para baixo para ver o card.
🤖 Com IA (automático)
1. Toque em "📷 Câmera NF"
2. Aponte para o DANFE ou NF
3. O sistema detecta e captura automaticamente
4. A IA extrai o número e o fornecedor pelo CNPJ
5. Confira os dados — corrija se necessário
⌨️ Manual (se IA falhar)
1. Digite o número da NF no campo de texto
2. Toque no campo Fornecedor e selecione da lista suspensa
3. Se o fornecedor não estiver na lista, avise o administrador
✅ O botão "Registrar Saída" só fica disponível quando placa, lacre, NF e fornecedor estiverem preenchidos.

Situações e o que Fazer
Guia rápido para cada situação que pode aparecer no sistema
✅ Aprovado
O sistema encerra automaticamente. Continue a operação normalmente.
🚨 Desvio Detectado
Avise o responsável. Não mova a carga. O administrador irá confirmar com justificativa.
⚠️ Placa não autorizada
Confira se digitou sem hífen. Se estiver certo, peça ao administrador para cadastrar a placa.
⚠️ Lacre não cadastrado
Verifique o número fisicamente no lacre e tente a entrada manual. Se correto, peça ao admin para cadastrá-lo.
⚠️ Processo aberto para essa placa
Já existe uma viagem em andamento para essa placa. A anterior precisa ser encerrada antes de registrar nova saída.
⚠️ Lacre em uso
Esse lacre já está sendo usado em outra viagem. Use um lacre diferente e comunique o administrador.
Dúvidas Frequentes
Respostas rápidas para as situações mais comuns no dia a dia
A câmera não abre. O que fazer? ▾
Android: quando o navegador pedir permissão, toque em "Permitir". Se já negou: Configurações → Chrome → Permissões → Câmera → encontre o endereço do app e permita. iPhone: Ajustes → Safari → Câmera → "Permitir".
A IA não leu o lacre. O que faço? ▾
Um campo de entrada manual aparece automaticamente. A foto já está registrada. Basta digitar o número que está impresso fisicamente no lacre e confirmar.
Registrei com a placa errada. E agora? ▾
Contate o administrador. Ele encerra o processo incorreto com justificativa e você registra novamente com a placa certa.
A placa aparece como "não autorizada". ▾
Verifique se digitou sem hífen e sem espaços (ex: ABC1234). Se a placa estiver certa, ela precisa ser cadastrada pelo administrador.
O lacre aparece como "não cadastrado". ▾
A IA pode ter lido errado — tente a entrada manual com o número exato impresso no lacre. Se correto, contate o administrador para cadastrá-lo.
Estou sem internet. Consigo usar? ▾
Sim, por 7 dias após o último login online. Uma faixa laranja aparece no topo indicando o modo offline. Os dados são sincronizados quando a internet voltar.
A sessão expirou e fui deslogado. É normal? ▾
Sim. O sistema faz logout automático após 2 horas de inatividade por segurança. Qualquer toque na tela reinicia o contador.
Troquei de celular. Perdi os dados? ▾
Não. Todos os dados ficam na nuvem. Basta acessar com o mesmo e-mail e senha no novo celular.
O app não atualiza — está mostrando versão antiga. ▾
Android: no Chrome, toque nos 3 pontinhos → "Atualizar página". Se não resolver, acesse o link direto no Chrome, atualize e reinstale o atalho. iPhone: no Safari, puxe a página para baixo para recarregar.
Já tem um processo aberto para essa placa. ▾
Cada placa só pode ter um processo aberto por vez. A viagem anterior precisa ser encerrada antes de registrar nova saída. Contate o administrador.
